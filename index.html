Customer Rate Explained
When this feature is enabled on the LANE ANALYSIS page, the Margin/Markup percentage is displayed here as "Margin/Markup" along with the rate that results from adding that Margin/Markup onto the rate - noted as "Customer Rate".
  • The Linehaul/All-In toggle controls whether the percentage is added onto the Linehaul rate or the All-In rate.
  • The Historical/Forecasted toggle controls whether the percentage is added to the historical rate under the "Lane Analysis" section or the forecasted rate under the "Forecasting" section.
    • See the help for Margin/Markup on the LANE ANALYSIS page for further details on how to turn on and control the Margin/Markup.
  • Margin is the percentage/dollars difference between the selling price and the desired revenue, or Customer Rate = Analysis Result/(1 – Margin). For example, if a lane analysis returns $1,000 All-In Rate and a 25% Margin is desired, the resulting "Customer Rate" would be $1,000/(1 – .25) = $1,333.33.
  • Markup is a simple percentage added to the All-In Rate. For example: if a lane analysis returns $1,000 All-In Rate and is marked up 25%, the resulting "Customer Rate" would be $1,000 x 1.25 = $1,250.
